# sre-interview
1. Go to https://killercoda.com/playgrounds/scenario/kubernetes and sign in.
2. Clone this repo in your Killercoda environment.
3. Apply the manifest `deployment.yaml` to the Kubernetes cluster.
4. Observe the pod status and health
5. Debug any issues.
6. Fix any issues.
7. Re-apply and verify the pod is Running.
8. Commit your fix and push to your fork (or just show the commit).
